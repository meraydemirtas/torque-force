# Seesaw Simulation

Basit bir tahterevalli simülasyonu. Canvas kullanarak yapıldı.

## Nasıl Kullanılır

Sayfayı açın ve tahterevallinin üzerine tıklayın. Rastgele ağırlıklarda (1-10 kg) nesneler düşecek. Sol ve sağ taraftaki toplam ağırlıkları görebilirsiniz.

Reset butonuna tıklayarak baştan başlayabilirsiniz.

## Özellikler

- Canvas ile çizim
- Tıklama ile nesne ekleme
- Ağırlık hesaplama
- Otomatik denge
- LocalStorage ile kayıt

## Teknolojiler

HTML, CSS, JavaScript
